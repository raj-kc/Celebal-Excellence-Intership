# Shopping Dataset Analysis

## Project Overview

This project performs data exploration, cleaning, feature engineering, and visualization on a shopping dataset using Python and Pandas.

## Objectives

* Load and explore the dataset
* Handle missing values
* Remove duplicate records
* Convert price columns to numeric format
* Create new features such as:

  * Price Difference
  * Popularity Score
* Perform exploratory data analysis (EDA)
* Generate visual insights
* Export a cleaned dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Insights

* Most products belong to the Mid-Range and Premium price tiers.
* Product ratings are generally consistent across all price categories.
* Price and rating show a very weak correlation.
* Popularity score helps identify highly trusted products.
